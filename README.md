## Run Locally

Clone the project

```bash
  git clone https://github.com/NVTIVZ/Twitter-Typescript
```

Go to the project directory

```bash
  cd Twitter-TypeScript
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
